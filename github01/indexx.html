GitHub using step

（1）在Github上建立一個新的專案



（2）在儲存庫名稱的地方填寫「自己設的名稱」.github.io，勾選public之後點選create repository



（3）跳轉到此網頁代表成功建立專案。點擊uploading an existing file，即可上傳檔案



（4）點選choose files上傳檔案，或直接拖曳資料夾至方框



（5）出現此畫面代表上傳成功



（6）點選commit changes檔案就會儲存在資料夾中



（7）點選畫面中的setting



（8）點選pages，將Build and deployment變更為GitHub Actions，再點擊Static HTML的configure架設靜態網頁



（9）點擊兩次commit change後，靜態網頁即架設成功


（10）再次點擊settings和pages，可以看到我們的網址已經成功上傳到網路上。再按visit site



（11）點選後出現這個畫面，此時顯示的還是File not found，不用緊張！



（12）我們需要在網址的後面再加上資料夾和檔案名稱



（13）更改後就網頁就會成功出現了！
